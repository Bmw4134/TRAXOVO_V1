Import Asset-Driver Relationships

Select an Excel file containing asset-driver mappings. The system will look for a sheet named "DRIVERS".
Back to List
File Format Information

The system expects an Excel file with the following format:

  • A sheet named "DRIVERS" containing asset-driver assignments
  • Columns for Asset ID, Driver Name/Employee Name, and Employee ID
  • The monthly equipment billings spreadsheet typically has this format
Note:

This import will update existing driver-asset relationships and create new ones as needed. Existing drivers will be matched by name or ID.

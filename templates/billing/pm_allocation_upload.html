How It Works
1. Upload Files

Upload both the original PM allocation file and the updated version with any changes or adjustments made. The system supports Excel (.xlsx, .xls) and CSV file formats.

2. Automatic Comparison

The system automatically compares the two files, identifying changes in allocation amounts, new assets, and removed assets. It calculates the total difference and percentage changes for each asset.

3. Review Results

The comparison results are displayed in an interactive report, highlighting increases, decreases, and unchanged allocations. You can export the results for further review or record-keeping.

Important: Make sure both files have the same structure and asset identifiers for accurate comparison. The system will attempt to match assets across files based on their IDs or descriptions.

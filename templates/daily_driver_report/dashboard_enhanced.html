How It Works
1. Upload Data Files

Upload GPS tracking data from the Gauge API to provide driver location information. The system supports multiple file formats and automatically detects file types.

2. Automatic Processing

Our GENIUS CORE engine processes all data sources, matching drivers to job sites and analyzing start/end times against established rules.

3. Driver Classification

Drivers are automatically classified into categories: On Time, Late Start, Early End, or Not On Job. The system captures exact times and calculates minutes late/early.

4. Report Generation

A complete Excel report is generated in the exact format needed for distribution, with summary statistics and detailed records for each driver.

Time-Saving Automation

What used to take 1-2 hours of manual work each day is now fully automated. The system produces the same results with consistent classification rules and complete traceability to source data files.

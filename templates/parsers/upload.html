Processing Information

SYSTEMSMITH uses advanced adaptive logic to process uploaded files:

  1. Format Detection: Automatically identifies file type and format
  2. Sheet Selection: Intelligently selects the most relevant sheet in multi-sheet Excel files
  3. Column Mapping: Maps columns to standardized field names for consistent processing
  4. Data Normalization: Cleans and normalizes data for analysis
  5. Employee Matching: Matches assets to employees using pattern recognition

The Project Manager (PM) review process:

  1. Initial Upload: Upload the original file for processing
  2. Download Results: Review and download the processed data
  3. PM Adjustments: Make necessary adjustments in the downloaded file
  4. Re-Upload: Upload the adjusted file for final processing
  5. Verification: The system will compare the original and adjusted files
  6. Approval: Confirm the changes to finalize the data

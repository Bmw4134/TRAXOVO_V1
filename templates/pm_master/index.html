About PM Master Billing

The PM Master Billing module allows you to batch process multiple PM allocation files at once, comparing them against a base file and generating a consolidated report.

This is ideal for handling month-end billing reconciliation across multiple regions or departments.

How It Works
  1. Select one or more allocation files (or use "Process All")
  2. The system will automatically identify the base file (labeled "FINAL REVISIONS")
  3. Each file is compared against the base file
  4. A consolidated report is generated with all discrepancies highlighted
Tips
  • The file with "FINAL REVISIONS" in the name is automatically used as the base file
  • If no base file is found, the first file is used as the base
  • The consolidated report contains summary and detailed data
  • Changes are highlighted for easy identification

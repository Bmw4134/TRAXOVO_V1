How It Works
1. Upload Data Files

Upload GPS tracking data from the Gauge API to provide driver location information. The system supports multiple file formats and automatically detects file types.

2. Automatic Processing

Our GENIUS CORE engine processes all data sources, matching drivers to job sites and analyzing start/end times against established rules.

3. Driver Classification

Drivers are automatically classified into categories: On Time, Late Start, Early End, or Not On Job based on GPS data and timecard information.

4. Report Generation

A comprehensive report is generated with metrics, detailed driver information, and exportable formats for sharing with management.

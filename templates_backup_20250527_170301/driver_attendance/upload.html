Required Format

Your files must include these columns:

  • Driver ID - Unique driver identifier
  • Date - Attendance date (YYYY-MM-DD)
  • Check-in - Start time
  • Check-out - End time
  • Location - Work location (optional)
Sample Data Format
Driver ID Date Check-in Check-out Location
D001 2025-05-27 08:00 17:00 DFW-001
D002 2025-05-27 07:30 16:30 HOU-002
Tips
  • Leave Check-in empty for absent drivers
  • Dates can be in various formats (Excel dates, MM/DD/YYYY, etc.)
  • Multiple files will be processed together
  • Duplicate driver records on the same date will be merged
